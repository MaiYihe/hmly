这是黑马旅游（后端）个人跟练项目
- 这是一个主要针对 Elasticsearch 的实战项目

技术栈：

JDK 1.8 + SpringBoot 2.x + Elasticsearch 8.x + MybatisPlus
- ES 集合 Kinbana 与 IK 分词器使用

创建索引库（指定字段的 Mapping）

根据索引 id 删除索引库

根据索引 id 查询索引库

新增文档、批量新增文档

根据索引名与文档 id 查询文档

根据索引名与文档 id 删除文档
